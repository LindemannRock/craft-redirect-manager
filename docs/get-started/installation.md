# Installation

## Via Composer

```bash
cd /path/to/project
```

```bash
composer require lindemannrock/craft-redirect-manager
```

```bash
./craft plugin/install redirect-manager
```

## Using DDEV

```bash
cd /path/to/project
```

```bash
ddev composer require lindemannrock/craft-redirect-manager
```

```bash
ddev craft plugin/install redirect-manager
```
