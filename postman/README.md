# Redirect Manager — Postman Files

User-facing Postman collection and environment template for the Redirect Manager JSON API.

- **`Redirect-Manager.postman_collection.json`** — example requests for the read-only redirects endpoint plus token, JSON, and rate-limit checks.
- **`Redirect-Manager.postman_environment.json`** — reusable environment template with placeholders only (no real token ships in the file).

## Documentation

Full setup, environment variables, and recommended test flows are documented here:

- [Postman Collection](../docs/developers/postman.md)
- [JSON API endpoint](../docs/developers/api-endpoints.md)

You can also download these files from the Control Panel: **Redirect Manager → Settings → Test → Download Postman collection**.
